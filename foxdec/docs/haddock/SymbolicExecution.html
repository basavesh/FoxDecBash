Do predicate transformation over a basic block in a CFG. Given an edge in the CFG from blockId to blockId', perform predicate transformation. blockId' is needed to set the flags properly. If none is supplied, the flags are ignored. We provide the option to skip the last instruction, as sometimes we need to symbolically execute a block up to but excluding the last instruction.

Tries to guess if the symbolic expression is a pointer. Any immediate that falls in the address range of the binary is highly likely a pointer. A value stored in a memory location belonging to an external function symbol is also highly likely a pointer.
