type Annot = [(Word64, NASM_Label, Word64)] Source #

An annotation consists of an address that is being symbolized to a label and an offset. The offset will often be 0. For example: 0x1016 --> L1000_2 0x4028 --> L_.bss_0x4020 + 0x8
